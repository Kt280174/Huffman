### Словарь:
Каждый символ (байт) сохраняется вместе с его длиной кода и его двоичным кодом.
Например:
Для символа A с кодом 101, данные будут сохранены как:
65 (ASCII-код 'A') | 3 (длина кода) | 5 (значение двоичного числа 101)
